Superstore Data Cleaning and Visualization

Objective

To clean and analyze the Superstore sales dataset using Python and create visualizations to identify business insights.

Dataset

Sample Superstore Dataset

Tasks Performed

- Checked missing values
- Removed duplicate records
- Detected outliers using boxplots
- Created data visualizations

Tools Used

- Google Colab
- Pandas
- Matplotlib
- Seaborn

Visualizations

1. Sales by Category
2. Top 10 States by Sales
3. Sales Distribution

Key Findings

- Missing values were analyzed and handled.
- Duplicate records were removed.
- Outliers were identified in Sales and Profit.
- Sales performance varied across categories and states.
- The dataset was successfully cleaned and visualized.
